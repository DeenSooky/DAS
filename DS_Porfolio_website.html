  • Designing a thrust reverser actuation system lock (TRASL) using Solidworks

    Insert a brief description of yourself here.

    Insert a brief description of yourself here.

    Designed a 3D model of a Thrust Reverser Actuation System Lock (TRASL) that had been carefully designed to achieve versatility and compatibility with both locking actuators and the third lock. The TRASL device had been engineered to withstand even the most challenging scenarios, with the maximum torque set at a robust 120 Nm in the event of a failure. The total weight of the TRASL, without fluids, is targeted at 3 kg, with a maximum limit of 3.5 kg to maintain the unit's manoeuvrability. The reserve stress factors had been set at a minimum of 1.5, providing an extra layer of safety for the user. For ease of maintenance, the TRASL unit could be attached to the Locking Actuators and Third Lock using M6 x 20 Aerospace Waspalloy fasteners, with the minimum size of any other fasteners set at M5 for added stability.

  • Desgining intelligent agents Project

    Insert a brief description of yourself here.

    The code you are looking at manages the creation and operation of a scene display window. The initialise function creates the window, while the register function creates a number of lights and robots, dividing them into two lists: "registerActives" for things that move (e.g. robots) and "registerPassives" for things that don't move (e.g. lights, charging stations, barriers). The code runs an endless loop through the moveIt function, which calculates the light sensed by each robot using the "senseLight" method in the Bot class, adjusts the speed of the motors based on the information gathered, using the "brain" method in the Bot class, and updates the state of the robot by moving it a small step using the "move" method in the Bot class. The code pauses for 50 milliseconds and repeats this process using the tkinter "after" function. The mathematics behind the robot movement is taken from the book "Computational Principles of Mobile Robotics" by Dudek and Jenkin (2011) published by Cambridge University Press.
